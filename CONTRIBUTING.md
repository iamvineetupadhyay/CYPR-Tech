# Contributing to CYPR Tech

First off, thank you for considering contributing to CYPR Tech! It's people like you who make this platform amazing.

## Code of Conduct

By participating in this project, you agree to abide by our [Code of Conduct](CODE_OF_CONDUCT.md).

## How Can I Contribute?

### Reporting Bugs
- Search existing issues to see if the bug has already been reported.
- If not, open a new issue using our bug report template.
- Include a clear title, description, steps to reproduce, and environment details (OS, JDK version, Browser).

### Suggesting Enhancements
- Check existing issues/discussions.
- Open an enhancement suggestion issue detailing the proposed change and its benefits.

### Pull Requests
1. Fork the repository and create your branch from `main`.
2. Write clean code adhering to Google Java Style or standard JS practices.
3. Test your changes locally (`mvn test` for backend).
4. Reference related issues in your commit messages.
5. Submit a Pull Request matching our PR template.

## Development Setup
- Backend: Java 17 + Spring Boot 3.3 + PostgreSQL
- Frontend: HTML5 Static / Next.js 14
