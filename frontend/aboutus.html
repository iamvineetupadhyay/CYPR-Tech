About CYPR

Cyber Protection & Threat Intelligence

Welcome to CYPR, an open-source, automated threat intelligence and security analysis platform designed to democratize web safety and digital asset protection.

In an era where malicious web infrastructure, phishing campaigns, and malware deployment loops evolve at an unprecedented pace, standard reactive defenses fall short. CYPR bridges the gap between raw global threat data and actionable real-time defense mechanisms by providing immediate, high-throughput verification vectors for URLs and files.

Why CYPR?

Intentions & Mission of the Project

01

Proactive Threat Mitigation

Moving away from traditional periodic checks to a continuous, proactive parsing system that neutralizes attack surfaces before they interact with users.

02

High-Throughput Ingestion

Building automated pipelines capable of handling massive volumes of open-source threat feeds without introducing system bottlenecks or high latency.

03

Accessible Automation

Providing developers, organizations, and everyday users with a unified, clean interface to instantly verify suspicious hashes, domains, and scripts through multi-engine analysis.

1. Cloud Infrastructure & Reverse Proxy

  • Architected and deployed the live production-grade backend environment hosted on an AWS EC2 standalone instance.
  • Configured and optimized an isolated managed AWS RDS PostgreSQL database cluster to securely isolate system state records and transaction metadata.
  • Implemented an Nginx Reverse Proxy integrated with dynamic DuckDNS configurations to securely route public API endpoints, while engineering programmatic filters to eliminate cross-origin (CORS) preflight blocks between our Vercel client and AWS server layers.
🚀

2. The VAJRA Core Threat Ingestion Engine

  • Designed and engineered VAJRA, a high-velocity concurrent ingestion module within Spring Boot.
  • VAJRA implements asynchronous background scheduling tasks that systematically pull, sanitize, and ingest over 1.42 Lakh+ active malicious domains from global threat intelligence networks (such as OpenPhish and Phishing Army) directly into optimized in-memory caches upon server initialization, delivering sub-millisecond look-up times.
🛡️

3. Multi-Engine Scans & Anti-Abuse Protocols

  • Integrated multi-source signature verification layers by orchestrating parallel asynchronous requests to the VirusTotal API, aggregating results dynamically from over 90+ global security engines.
  • Hardened the application perimeter by embedding Spring Security filter chains with custom stateless JSON Web Tokens (JWT) and enterprise-grade GitHub OAuth 2.0 authentication mechanisms.
  • Mitigated automated bot scripts and script-driven API abuse by developing server-side verification routines for Cloudflare Turnstile CAPTCHA.
  • Configured synchronous transactional alerts via Brevo SMTP engines to dispatch diagnostic payloads and incident response notifications directly to registered user targets.
